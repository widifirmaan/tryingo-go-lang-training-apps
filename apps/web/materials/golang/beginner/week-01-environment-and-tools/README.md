# Week 1: Environment & Tools

## Learning Objectives
- Understand what Go is and why it's popular
- Install Go on your operating system
- Set up a development environment (VS Code)
- Write and run your first Go program
- Understand Go modules and packages

## Project: Hello World CLI
Build a command-line program that prints a personalized greeting.

## Key Takeaways
- Go is compiled, statically typed, and garbage-collected
- Programs start from `func main()` in `package main`
- Use `go run` for quick execution, `go build` for production binaries
- Go modules manage dependencies and versions
