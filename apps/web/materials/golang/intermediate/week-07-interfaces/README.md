# Week 7: Interfaces

## Overview
Interfaces are Go's way of defining behavior through method sets. They enable polymorphism and flexible, decoupled code.

## Objectives
- Understand interface types and method sets
- Implement interfaces implicitly
- Use empty interfaces for dynamic typing
- Apply type switches and assertions
- Design clean, composable interfaces

## Weekly Project: Shape Calculator
Build a program that calculates area and perimeter for various shapes (Circle, Rectangle, Triangle) using interfaces, then extends to 3D shapes with volume.
