# Week 8: Error Handling

## Overview
Go treats errors as values. This week covers creating, wrapping, inspecting, and recovering from errors.

## Objectives
- Understand the error interface
- Create custom error types
- Wrap errors with additional context
- Handle panics with recover
- Use the errors utility package

## Weekly Project: File Backup System
Build a backup tool that copies files with comprehensive error handling: validation, permission checks, disk space checks, partial failures, and detailed error reporting.
