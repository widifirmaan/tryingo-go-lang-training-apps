# Week 9: Packages & Modules

## Overview
Go uses packages for code organization and modules for dependency management. This week covers structuring, exporting, and managing Go projects.

## Objectives
- Create and organize packages
- Understand exported vs unexported identifiers
- Use internal packages for encapsulation
- Manage dependencies with `go.mod`
- Publish and version modules

## Weekly Project: Multi-Package CLI Tool
Build a CLI todo application organized into `cmd`, `internal`, and `pkg` packages with proper dependency management.
