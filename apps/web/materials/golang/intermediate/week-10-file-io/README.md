# Week 10: File I/O

## Overview
Go provides powerful primitives for reading, writing, and manipulating files and directories using the `os`, `io`, and `bufio` packages.

## Objectives
- Read files using multiple methods
- Write files with different strategies
- Use buffered I/O for performance
- Navigate and manage directories
- Handle file permissions and errors

## Weekly Project: Markdown Note Manager
Build a CLI tool that creates, lists, reads, and archives Markdown notes with directory organization, search, and backup features.
