# Week 11: JSON & Encoding

## Overview
Go has first-class support for JSON and other data formats through the `encoding/json` and `encoding` packages.

## Objectives
- Marshal Go types to JSON
- Unmarshal JSON into Go types
- Customize encoding with struct tags
- Implement custom Marshal/Unmarshal logic
- Encode/decode other formats (XML, CSV, YAML)

## Weekly Project: REST API Client
Build a JSON API client that fetches, parses, and transforms data from a RESTful service with custom encoding and format conversion.
