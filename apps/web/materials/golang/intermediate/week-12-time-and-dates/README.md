# Week 12: Time & Dates

## Overview
Go's `time` package provides a rich set of functions for working with time, durations, scheduling, and formatting.

## Objectives
- Create and manipulate time values
- Format and parse time strings
- Work with durations and time zones
- Use timers and tickers for scheduling
- Compare and calculate time differences

## Weekly Project: Event Scheduler
Build a command-line calendar tool that creates events, checks for conflicts, sends reminders, and displays schedules in different time zones.
