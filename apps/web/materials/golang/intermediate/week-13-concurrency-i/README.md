# Week 13: Concurrency I

## Overview
Go's concurrency model is built on goroutines and channels. This week covers spawning goroutines, communicating via channels, and synchronizing with WaitGroups.

## Objectives
- Create and manage goroutines
- Use channels for communication
- Synchronize with WaitGroup
- Apply common channel patterns
- Avoid race conditions

## Weekly Project: Web Crawler
Build a concurrent web crawler that fetches pages in parallel, respects rate limits, reports progress, and collects results.
