# Week 16: Testing I

## Overview
Go has a built-in testing framework. This week covers writing tests, table-driven tests, coverage analysis, and benchmarking.

## Objectives
- Write and run unit tests
- Use table-driven test patterns
- Measure and improve coverage
- Write and run benchmarks
- Test helper functions

## Weekly Project: Calculator Test Suite
Build a comprehensive test suite for a calculator library with unit tests, table tests, edge case coverage, and performance benchmarks.
