# Week 28: Message Queues

Learn message queue concepts and implement messaging with NATS, RabbitMQ, and Kafka in Go.

## Learning Objectives
- Understand queue-based messaging patterns
- Implement pub/sub and request/reply patterns
- Compare NATS, RabbitMQ, and Kafka
- Handle message persistence and delivery guarantees

3. [RabbitMQ](03-rabbitmq.md)
