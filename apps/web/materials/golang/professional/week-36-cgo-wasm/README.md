# Week 36: CGo and WebAssembly

Interoperate with C libraries using CGo and compile Go to WebAssembly.

## Learning Objectives
- Call C code from Go using CGo
- Build CGo-enabled applications
- Compile Go to WebAssembly
- Interact with browser DOM from Go WASM

- Build a WASM module in Go
