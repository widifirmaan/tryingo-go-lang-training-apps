# Week 9: Advanced Forms

## Overview
HTML5 introduces powerful form features that go beyond basic input fields. This week covers advanced validation techniques, the datalist element, new form attributes, and building custom form controls.

## Learning Objectives
- Implement client-side validation using HTML5 constraints
- Use the datalist element for autocomplete suggestions
- Leverage modern form attributes like `autocomplete`, `novalidate`, and `form`
- Build custom form controls with JavaScript

## Weekly Project
**Smart Registration Form** — Build a registration form with real-time validation, password strength meter, country picker with datalist, and custom select menus.

## Topics
1. Form Validation (constraint validation API, patterns, error messages)
2. Datalist and Autocomplete
3. New Form Attributes
4. Custom Form Controls
