# Week 11: SEO (Search Engine Optimization)

## Overview
Search Engine Optimization ensures your content ranks well in search results. This week covers meta tags, structured data, sitemaps, and SEO best practices for HTML5.

## Learning Objectives
- Write effective meta tags for search engines and social media
- Implement structured data with JSON-LD (Schema.org)
- Create and submit XML sitemaps
- Apply technical SEO best practices

## Weekly Project
**SEO-Optimized Product Page** — Create a product page with complete meta tags, structured data for products, breadcrumbs, and reviews.

## Topics
1. Meta Tags for SEO and Social
2. Structured Data (JSON-LD / Schema.org)
3. XML Sitemaps and robots.txt
4. Technical SEO Best Practices
