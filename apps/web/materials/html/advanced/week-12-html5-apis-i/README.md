# Week 12: HTML5 APIs I

## Overview
HTML5 introduced powerful client-side APIs. This week covers Geolocation, Drag & Drop, Web Storage, and IndexedDB — enabling rich, data-aware web applications.

## Learning Objectives
- Access user location with the Geolocation API
- Implement drag-and-drop interactions
- Use Web Storage (localStorage/sessionStorage) effectively
- Store structured data client-side with IndexedDB

## Weekly Project
**Location-Based Notes App** — Build an app that stores geotagged notes using IndexedDB, allows drag-and-drop reordering, and persists data across sessions.

## Topics
1. Geolocation API
2. Drag and Drop API
3. Web Storage (localStorage/sessionStorage)
4. IndexedDB
