# Week 13: HTML5 APIs II

## Overview
This week covers graphics and animation APIs: Canvas for pixel-level drawing, SVG for scalable vector graphics, CSS animations, and the Web Animations API.

## Learning Objectives
- Draw graphics programmatically with Canvas
- Create and manipulate SVG elements
- Build CSS animations and transitions
- Use the Web Animations API for JavaScript-driven animations

## Weekly Project
**Interactive Data Dashboard** — Build a dashboard with canvas charts, SVG icons, animated transitions, and a real-time data visualization.

## Topics
1. Canvas API
2. SVG (Scalable Vector Graphics)
3. CSS Animations
4. Web Animations API
