# Week 14: HTML5 APIs III

## Overview
This week covers advanced browser APIs for managing history, running background scripts, enabling offline support, and building resilient web applications.

## Learning Objectives
- Manipulate browser history with the History API
- Run background tasks with Web Workers
- Implement service workers for offline support
- Build offline-first web applications

## Weekly Project
**Offline-First News Reader** — Build a news reader that caches articles via service worker, uses web workers for content processing, and works fully offline.

## Topics
1. History API
2. Web Workers
3. Service Workers
4. Offline Support
