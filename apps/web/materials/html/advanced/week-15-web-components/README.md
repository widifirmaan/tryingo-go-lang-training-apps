# Week 15: Web Components

## Overview
Web Components are a set of browser APIs for creating reusable, encapsulated custom HTML elements. This week covers Custom Elements, Shadow DOM, HTML Templates, and component patterns.

## Learning Objectives
- Define and use custom HTML elements
- Encapsulate styles and markup with Shadow DOM
- Use HTML templates for reusable content
- Apply component design patterns

## Weekly Project
**Custom UI Component Library** — Build a library of reusable web components: custom button, tooltip, modal, accordion, and tabs.

## Topics
1. Custom Elements
2. Shadow DOM
3. HTML Templates and Slots
4. Component Design Patterns
