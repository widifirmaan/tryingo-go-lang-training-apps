# Week 16: Modern HTML

## Overview
Modern HTML introduces new elements and APIs that simplify common UI patterns. This week covers the `<dialog>` element, `<details>`/`<summary>`, `<search>`, and modern `<datalist>` usage.

## Learning Objectives
- Use the `<dialog>` element for modals and popups
- Create expandable content with `<details>` and `<summary>`
- Implement semantic search with `<search>` element
- Combine modern elements for rich UI patterns

## Weekly Project
**Modern Component Playground** — Build an interactive showcase using `<dialog>`, `<details>`, `<search>`, and `<datalist>` in a single-page app.

## Topics
1. Dialog Element and API
2. Details and Summary
3. Search Element
4. Advanced Datalist Patterns
