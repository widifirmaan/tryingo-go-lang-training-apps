# Week 17: Modern CSS

## Overview
Modern CSS adds powerful features for better organization and maintainability. This week covers container queries, CSS layers, nesting, and custom properties.

## Learning Objectives
- Write responsive components with container queries
- Manage CSS specificity with `@layer`
- Use native CSS nesting for cleaner stylesheets
- Leverage CSS custom properties for dynamic theming

## Weekly Project
**Themeable Component Library** — Build a set of themeable UI components using CSS custom properties, container queries, layers, and nesting.

## Topics
1. Container Queries
2. CSS Layers (@layer)
3. CSS Nesting
4. CSS Custom Properties
