# Week 18: CSS Animations

## Overview
Advanced CSS animation techniques including transitions, keyframes, scroll-driven animations, and performance optimization.

## Learning Objectives
- Create smooth CSS transitions
- Build complex keyframe animations
- Implement scroll-driven animations
- Optimize animation performance

## Weekly Project
**Animated Landing Page** — Build a landing page with scroll-triggered animations, parallax effects, smooth transitions, and performant animations.

## Topics
1. Advanced Transitions
2. Keyframe Animations
3. Scroll-Driven Animations
4. Animation Performance
