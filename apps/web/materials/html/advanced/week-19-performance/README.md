# Week 19: Web Performance

## Overview
Web performance optimization ensures fast loading and interaction. This week covers lazy loading, critical CSS, resource hints, and Core Web Vitals.

## Learning Objectives
- Implement lazy loading for images, iframes, and scripts
- Extract and inline critical CSS
- Use resource hints (preload, prefetch, preconnect)
- Optimize for Core Web Vitals (LCP, CLS, INP)

## Weekly Project
**Performance-Optimized E-Commerce Page** — Optimize a product listing page for LCP, CLS, and INP using latest performance techniques.

## Topics
1. Lazy Loading
2. Critical CSS
3. Resource Hints
4. Core Web Vitals
