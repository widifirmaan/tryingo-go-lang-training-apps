# Week 20: Progressive Web Apps (PWA)

## Overview
PWAs combine the best of web and native apps. This week covers manifest files, service workers, caching strategies, and advanced PWA features.

## Learning Objectives
- Create and configure a web app manifest
- Implement service workers for offline support
- Apply appropriate caching strategies
- Add native-like features (install prompt, push notifications)

## Weekly Project
**PWA News Reader** — Convert a news website into a fully functional PWA with offline reading, push notifications, and install prompt.

## Topics
1. Web App Manifest
2. Service Worker Lifecycle
3. Caching Strategies
4. PWA Features (Install, Push, Sync)
