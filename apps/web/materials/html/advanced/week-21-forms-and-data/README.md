# Week 21: Forms and Data

## Overview
Advanced form handling beyond basic HTML5 validation — working with FormData, the Validation API, custom validation patterns, and file handling.

## Learning Objectives
- Use FormData API for programmatic form handling
- Master Constraint Validation API for custom validation
- Build complex custom validation patterns
- Handle file uploads with preview and progress

## Weekly Project
**Advanced Form Builder** — Build a dynamic form builder with custom validation rules, file upload with progress, and data export.

## Topics
1. FormData API
2. Constraint Validation API
3. Custom Validation Patterns
4. File Handling and Uploads
