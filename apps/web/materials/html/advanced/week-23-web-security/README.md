# Week 23: Web Security

## Overview
Web security is critical for protecting users and data. This week covers HTTPS, Content Security Policy, CORS, and XSS prevention.

## Learning Objectives
- Understand HTTPS and TLS fundamentals
- Implement Content Security Policy headers
- Configure CORS for cross-origin requests
- Prevent XSS, CSRF, and other injection attacks

## Weekly Project
**Secure Web Application** — Audit and secure a web application with CSP, CORS, input sanitization, and security headers.

## Topics
1. HTTPS and TLS
2. Content Security Policy (CSP)
3. CORS (Cross-Origin Resource Sharing)
4. XSS Prevention and Security Best Practices
