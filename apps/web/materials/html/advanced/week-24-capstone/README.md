# Week 24: Capstone Project

## Overview
The capstone project integrates all HTML5 advanced topics — forms, accessibility, SEO, APIs, Web Components, modern CSS, performance, PWA, testing, and security — into a single production-quality application.

## Learning Objectives
- Plan and architect a complex web application
- Implement all features using modern HTML5 techniques
- Optimize for performance, accessibility, and SEO
- Deploy a production-ready web application

## Weekly Project
**Complete Portfolio Platform** — Build a portfolio/career platform with project showcase, contact form, blog, and admin dashboard. Fully accessible, SEO-optimized, PWA-capable, and secure.

## Topics
1. Planning and Architecture
2. Implementation
3. Optimization
4. Launch and Deployment
