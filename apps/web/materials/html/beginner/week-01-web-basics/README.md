# Week 1: Web Basics

## Learning Objectives
- Understand how the internet and web work
- Learn what HTML is and its role in web development
- Create your first HTML page
- Set up browser DevTools and a code editor

## Lessons
1. [How the Internet Works](01-how-web-works.md) - DNS, HTTP, client-server model
2. [Introduction to HTML](02-html-intro.md) - What is HTML, tags, elements, attributes
3. [Your First Webpage](03-first-page.md) - Creating a complete HTML document
4. [Browser DevTools & Editors](04-tools.md) - Chrome DevTools, VS Code setup

## Project: Personal Introduction Page
Build a simple webpage that introduces yourself using basic HTML tags.

## Key Takeaways
- The web uses a client-server architecture with HTTP protocol
- HTML provides structure to web content using tags
- Every HTML page needs DOCTYPE, html, head, and body
- Browser DevTools are essential for debugging
