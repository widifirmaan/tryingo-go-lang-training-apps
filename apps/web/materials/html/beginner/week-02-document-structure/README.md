# Week 2: Document Structure

Understanding the foundational structure of an HTML5 document is critical for building well-formed, accessible, and SEO-friendly web pages. This week covers everything from the DOCTYPE declaration to semantic HTML elements that give meaning to your content.

## Learning Objectives

By the end of this week you will be able to:

- Write a complete HTML5 document skeleton
- Use the `lang` attribute correctly
- Configure meta tags for character encoding, viewport, and social sharing
- Distinguish between semantic and non-semantic HTML elements
- Organize body content with proper landmarks

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | DOCTYPE & html Element | `<!DOCTYPE html>`, `<html lang="...">`, document structure |
| 2 | The `<head>` Element | `<meta>`, `<title>`, `<link>`, `<script>`, OG tags |
| 3 | Semantic HTML | `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`, `<figure>` |
| 4 | Body Structure | `<div>` vs semantic elements, HTML comments, content organization |

## Prerequisites

Completion of Week 1 or familiarity with basic HTML syntax (tags, attributes, elements).

## Practice

Create a complete HTML document that includes a proper DOCTYPE, `lang` attribute, meta viewport tag, title, and uses at least four different semantic elements in the body.
