# Week 3: Text & Links

This week focuses on the building blocks of web content — headings, paragraphs, lists, quotations, and anchors. You will learn how to structure text hierarchically, create navigation, and use inline text tags to add meaning.

## Learning Objectives

By the end of this week you will be able to:

- Implement a proper heading hierarchy (h1-h6)
- Create internal, external, and bookmark links with correct attributes
- Build ordered, unordered, and description lists
- Use quotation and inline text elements semantically

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | Headings | `h1`-`h6`, hierarchy, SEO impact |
| 2 | Anchors | `<a>`, `href`, `target`, `rel`, bookmark links |
| 3 | Lists | `<ul>`, `<ol>`, `<li>`, nested lists, `<dl>`, `<dt>`, `<dd>` |
| 4 | Quotes & Inline Text | `<blockquote>`, `<q>`, `<cite>`, `<abbr>`, `<address>`, `<strong>`, `<em>`, `<mark>`, `<small>`, `<ins>`, `<del>`, `<sub>`, `<sup>` |

## Prerequisites

Completion of Week 2 or familiarity with HTML document structure.

## Practice

Build a one-page article about a topic of your choice that includes proper heading hierarchy, at least two list types, quotations, and inline text formatting.
