# Week 4: Images & Media

Learn how to embed images, video, audio, and iframes into HTML documents. This week covers responsive images, accessible media, and embedding third-party content.

## Learning Objectives

By the end of this week you will be able to:

- Embed images with proper `src`, `alt`, and sizing attributes
- Use the `<picture>` element for responsive images
- Add video and audio with controls and multiple sources
- Embed YouTube videos and other content using iframes

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | Images | `<img>`, `src`, `alt`, `width`, `height`, file formats |
| 2 | Responsive Images | `<picture>`, `<source>`, `srcset`, media queries |
| 3 | Video & Audio | `<video>`, `<audio>`, `<source>`, `controls`, `poster` |
| 4 | iframes | `<iframe>`, `srcdoc`, `sandbox`, embedding YouTube and maps |

## Prerequisites

Completion of Week 3 or familiarity with HTML tags and attributes.

## Practice

Build a media gallery page that includes a responsive image, an embedded video, and an iframe map.
