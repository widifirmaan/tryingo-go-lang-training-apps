# Week 5: Tables & Forms

Learn how to present tabular data with HTML tables and collect user input with forms. This week covers the full table structure, form submission methods, input types, and accessibility best practices.

## Learning Objectives

By the end of this week you will be able to:

- Build accessible HTML tables with `thead`, `tbody`, `tfoot`
- Use `colspan` and `rowspan` for complex table layouts
- Create forms with proper `action` and `method` attributes
- Choose the right input type for each data field
- Use `<label>`, `<fieldset>`, and `<legend>` for accessible forms

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | Tables | `<table>`, `<thead>`, `<tbody>`, `<tfoot>`, `<tr>`, `<th>`, `<td>`, colspan, rowspan |
| 2 | Form Basics | `<form>`, `action`, `method`, GET vs POST, form submission |
| 3 | Input Types | text, email, password, number, date, checkbox, radio, file, range, color, submit, hidden |
| 4 | Labels & Accessibility | `<label>`, `for`, `<fieldset>`, `<legend>`, ARIA attributes |

## Prerequisites

Completion of Week 4 or familiarity with HTML elements and attributes.

## Practice

Build a registration form with at least 6 different input types and a schedule table with merged cells.
