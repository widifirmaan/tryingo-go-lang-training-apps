# Week 6: CSS Basics

Cascading Style Sheets (CSS) control the visual presentation of HTML. This week covers CSS syntax, selectors, the box model, colors, typography, and backgrounds.

## Learning Objectives

By the end of this week you will be able to:

- Write CSS rules using element, class, ID, and attribute selectors
- Understand the cascade and specificity
- Apply the box model (content, padding, border, margin)
- Use named colors, hex, RGB, HSL, and opacity
- Style text with typography properties
- Add background colors, images, and gradients

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | CSS Introduction | Syntax, selectors, cascade, specificity |
| 2 | Box Model | Content, padding, border, margin, box-sizing |
| 3 | Colors & Typography | color formats, font properties, text properties |
| 4 | Backgrounds | background-color, image, gradient, shorthand |

## Prerequisites

Completion of Week 5 or familiarity with HTML elements and attributes.

## Practice

Style an HTML page using all four selector types, apply box model properties, and use at least two different color formats.
