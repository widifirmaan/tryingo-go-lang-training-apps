# Week 7: CSS Layout

Learn modern CSS layout techniques using Flexbox, CSS Grid, positioning, and responsive design patterns.

## Learning Objectives

By the end of this week you will be able to:

- Create flexible layouts with Flexbox
- Build two-dimensional layouts with CSS Grid
- Control element positioning with `position` property
- Build responsive designs with media queries and relative units

## Topics Covered

| Lesson | Topic | Key Concepts |
|--------|-------|--------------|
| 1 | Flexbox | `display: flex`, directions, alignment, flex-grow/shrink |
| 2 | CSS Grid | `display: grid`, columns/rows, gap, placement, auto-fit |
| 3 | Positioning | static, relative, absolute, fixed, sticky, z-index |
| 4 | Responsive Design | Media queries, mobile-first, breakpoints, relative units |

## Prerequisites

Completion of Week 6 or familiarity with CSS selectors and the box model.

## Practice

Build a responsive page layout that uses Flexbox for navigation, CSS Grid for a card layout, and includes a sticky header.
