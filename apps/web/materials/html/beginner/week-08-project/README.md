# Week 8: Final Project

Apply everything learned in Weeks 2-7 to build a complete, responsive multi-page website from scratch.

## Project Overview

You will plan, design, build, and deploy a multi-page HTML5 website using semantic HTML and CSS. The goal is to demonstrate mastery of document structure, text, media, forms, tables, CSS styling, and responsive layout.

## Project Options

Choose one of the following:

| Project | Description |
|---------|-------------|
| Personal Portfolio | Showcase your skills, projects, and contact information |
| Small Business Site | A landing page for a fictional business with services, about, and contact |
| Recipe Blog | A blog with recipe pages, ingredient tables, and a comment form |
| Event Landing Page | A conference or meetup page with schedule table, speaker cards, and registration |

## Requirements

- At least 3 HTML pages linked together
- Semantic HTML5 elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`)
- Proper `<head>` with meta tags, title, OG tags, and viewport
- A table (schedule, pricing, or data comparison)
- A form with at least 5 input types and proper labels
- Images with `alt` text (responsive where applicable)
- External CSS with proper selectors and the box model
- Flexbox and/or Grid layout
- Responsive design with at least 2 breakpoints
- CSS custom properties (variables)

## Submission Checklist

- [ ] Valid HTML5 (W3C validator passes)
- [ ] No inline CSS or JavaScript
- [ ] All images have `alt` text
- [ ] Form uses POST method and includes labels
- [ ] Page is responsive (mobile, tablet, desktop)
- [ ] Navigation works on all pages
- [ ] Code is indented and readable
- [ ] CSS in external stylesheet
