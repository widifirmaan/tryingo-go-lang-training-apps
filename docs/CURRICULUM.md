## How Content is Organized

Each module contains:
```
materials/{language}/{level}/
├── week-01-introduction/
│   ├── README.md              # Module overview
│   ├── 01-first-topic.md      # Lesson 1
│   ├── 02-second-topic.md     # Lesson 2
│   └── project.md             # Hands-on project
└── week-02-basics/
    └── ...
```

Each lesson includes:
1. Learning objectives
2. Theory with code examples
3. Interactive playground link
4. Practice exercises
5. Quiz questions
6. Summary
